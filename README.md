# Facebook Clone — React Native CLI + Firebase

A Facebook Clone built with React Native CLI (bare workflow) and Firebase.

## Features
- Email/Password Authentication
- Real-time News Feed
- Create Posts (Text)
- Like / Unlike Posts
- Comments
- Search Users
- Profile + Edit Profile
- Logout

## Tech Stack
- React Native CLI
- Firebase Auth
- Firebase Firestore
- React Navigation

## Setup
1. Clone the repo
2. Run `npm install`
3. Add your Firebase config in `src/firebase/config.js`
4. Run `npx react-native run-android`
